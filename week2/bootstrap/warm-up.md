## Exercises

#### Exercise 1

Center the following `wrapper` using CSS

```html
    <html>
        <head>
            <style>
                #wrapper {
                    /* what properties go here? */
                }
                p {
                    text-align: center;
                }
            </style>
        </head>
        <body>
            <section id="wrapper">
                <p>I am in the center!</p>
            </section>
        </body>
    </html>
```
